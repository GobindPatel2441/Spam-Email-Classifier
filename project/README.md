# Spam Email Classifier 

A beginner-friendly full-stack machine learning application that classifies custom text messages as **Spam** or **Not Spam**.

This project provides a straightforward approach to Natural Language Processing (NLP) with an easy-to-use HTML frontend and a Flask backend.

## 🎯 How It Works
1. **User Input:** The frontend `index.html` interface captures the email text.
2. **API Request:** The Javascript seamlessly sends a `POST` request to the backend.
3. **ML Prediction:** The Flask API passes the text to `predictor.py`. Data is cleaned (lowercased, stopwords removed via NLTK, stemmed), converted to Word Vector frequencies via TF-IDF, and run through the Multinomial Naive Bayes model.
4. **Response:** It returns the prediction ("Spam" or "Not Spam") and the system's exact confidence percentage.
5. **UI Display:** The Frontend updates dynamic UI cards cleanly based on the API response.

## 🧠 Why Naive Bayes?
Naive Bayes is heavily preferred for beginner text classification because:
- **Fast and Efficient:** Calculates strict conditional probabilities incredibly quickly comparing occurrences of "spammy" words relative to their normal uses.
- **Works well with Text:** Especially when leveraging TF-IDF Vectorization, treating each word as an isolated independent feature works remarkably well for determining Spam.

## 📁 Project Structure
```
project/
│── backend/
│   ├── app.py            → The Flask API Server
│   ├── model.py          → Training script logic to construct ML model
│   ├── predictor.py      → Helper function for inference (predict_spam)
│   ├── dataset.csv       → Data for initially training
│   ├── model.pkl         → The trained Naive Bayes weights (Generated by model.py)
│   ├── vectorizer.pkl    → The trained TF-IDF Vectorizer (Generated by model.py)
│
│── frontend/
│   ├── index.html        → The clean UI, integrated dynamically with our API
│
│── requirements.txt      → Required Python dependencies
│── README.md
```

## 🚀 How to Run

**1. Install Backend Dependencies**
```bash
pip install -r requirements.txt
```

**2. Train the Model**
If `model.pkl` and `vectorizer.pkl` don't exist yet, run this to train the model locally using the SMS Spam Collection dataset:
```bash
cd backend
python model.py
```

**3. Start the Flask Backend Server**
```bash
python app.py
```
*The server will start at http://localhost:5000*

**4. Open Frontend App**
Open `frontend/index.html` simply by double clicking the file in your Windows explorer. Enter some text and click "Check Spam"!
